Welcome!

Welcome to this introduction to computer science course. This course is meant teach the fundamentals of computer science and programming using the language of Java. If you are taking AP Computer Science, an introductory programming class in college, or you just want to get started with coding this is a great resource for you. You do not need to have any prior experience with coding to follow this course.


What You Will Learn

This course covers all the material that you would learn in the first semester of computer science class in college. We will first start with syntax, variables, and objects. We will then move into conditional statements, loops, and classes. Finally, we will end with error handling, sorting, searching, and recursion.


Advice for this Course

  • Don’t be discouraged: computer science is hard, and I struggled a lot when I first started learning. If you find yourself struggling, don’t think for one second that you should quit. Everyone can learn anything!
  • Learn by doing: this is especially true in computer science. Reading and understanding the theory is only one part of the learning. I would highly recommend applying your knowledge by writing code. Whether it is through the course exercises or a personal project the act of writing code will cement the concepts and build up your problem-solving skills.
  • Make use of the chatroom: if you are stuck with something feel free to use the chatroom to ask someone for help. I will be on as much as possible to help.
  • Help others: Explaining a concept to someone who is asking for help is a great way to make sure you really know the material. If you can teach it, you know it!
