Data Structures

A data structure is a way that information is stored in the computer’s memory. We are going to be manipulating and accessing this data, so storing our data in a certain way may give us a speed(time) or space(memory) advantage. Oftentimes how we store our data will be determined by what we are going to do most often with the data because we will optimize for whatever we are going to do most with the data.


What You Will Learn

In this course we are going to learn about the most common data structures as well as a few advanced ones. These will include lists, stacks, queues and deques, sets, trees, binary search trees, AVL and red-black trees, weighted and unweighted graphs, etc. We will also cover time complexity analysis and some of the algorithms that go along with these data structures: advanced sorting, Prim’s algorithm, Dijkstra’s algorithm, etc.
