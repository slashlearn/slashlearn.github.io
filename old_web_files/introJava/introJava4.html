Making Methods:

Here we see examples of two methods I have created

In the next two sections we are going to be creating our own methods, but before we do that let's go over some important parts of a method.

  • access specifier:This can be public or private. If it is public it means that the method can be used from outside the class, if it is private it means the method can only be called from other methods inside the same class
  • Return Type:The type of the value that is to be returned. In the picture the second method has the return type of void and this is used if nothing is being returned.
  • Method Name: The name of the method
  • Explicit Parameter(s): These are the arguments enclosed inside a methods parentheses, they are the arguments you may send to a method. You can have any amount of explicit parameters.
  • Method Body: This is where the implementation of the method takes place. In other words, this is where you write the code for what the method will do.

Exercise 4.1

Description: Congratulations! writing the Person class means you have officially written your first Java Class. Now you are going to expand on your knowledge of classes build another class by yourself.


Instructions:

  1. Create a class titled "Circle"
  2. Give the Circle class two instance variables a String called color and a double called radius.
  3. Create a default constructor which gives the color a default value of "Blue" and the radius the default value of 1
  4. Create a method called setColor and setRadius. These should set appropriate instance variables to values passed as explicit parameters. (These methods should not return anything)
  5. Create two methods called getColor and getRadius. These should return the appropriate instance variables.
  6. Create a method called getArea which will return the area of a particular circle as a double. (The area of a circle is pi * radius * radius, simply use 3.14 for pi value)
    Hint: area is not an instance field so in the method you will calculate the area and return it.
